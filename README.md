# Smart-Picker | 智能选购助手小程序

一个以视觉表现和交互体验为重点的微信小程序练习项目。

这个仓库展示了一个围绕商品浏览、加购、登录注册和个人中心展开的前端原型，适合作为微信小程序界面设计、交互动效和页面组织方式的参考项目。

## 项目亮点

- 高保真界面风格，强调卡片层次、半透明效果与沉浸式视觉体验
- 包含商品浏览、购物袋交互、登录注册弹窗和个人中心等页面模块
- 适合作为小程序前端原型或作品集展示项目继续扩展

## 项目结构

- `miniprogram/`：小程序前端代码
- `cloudfunctions/`：云函数目录
- `project.config.json`：微信开发者工具工程配置

## 已有功能

- 首页展示与列表浏览
- 商品卡片交互
- 登录 / 注册弹窗
- 个人中心页面
- 购物金额联动展示

## 本地运行

1. 克隆仓库到本地。
2. 使用微信开发者工具导入项目。
3. 打开仓库根目录并确认 `miniprogramRoot` 指向 `miniprogram/`。
4. 根据需要配置云开发环境后运行。
