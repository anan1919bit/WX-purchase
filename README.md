## 🌟 项目亮点

### 1. 像素级视觉还原 (Pixel-Perfect)
- **AI 协同重构**：基于业界高标准设计稿，利用 Codex 强大的代码生成能力，精准还原了复杂的 **毛玻璃（Glassmorphism）** 视觉效果与层级关系。
- **沉浸式 UI**：通过自定义 WXSS 实现了半透明高斯模糊 Header 和极具质感的商品卡片。

### 2. AI 驱动的开发模式
- **快速原型构建**：跳过繁琐的手写代码阶段，通过 **Prompt Engineering** 引导 AI 完成从页面布局到逻辑绑定的全过程。
- **工程化调试**：在 AI 生成代码的基础上，人工介入解决层级冲突、动画流畅度及逻辑闭环等关键细节。

### 3. 完整的交互逻辑
- **动态购物袋**：实现了商品数量实时增减反馈及底部状态栏金额自动计算。
- **模块化页面**：包含精选推荐流、高保真登录/注册弹窗以及个人数据中心。

## 🛠️ 技术栈
- **核心模型**：OpenAI Codex / DeepSeek-V3
- **辅助工具**：VS Code + Roo Code / Cline
- **前端技术**：微信小程序原生框架 (WXML / WXSS / JavaScript)

- [x] **精选首页**：高保真视觉渲染与列表滑动。
- [x] **交互组件**：加购动画反馈、毛玻璃导航栏。
